# Sudoku Solver Using Backtracking and Heuristics

This project implements a Sudoku solver using multiple approaches, including basic backtracking and optimized techniques using heuristics.

## Project Overview
This project demonstrates how algorithmic improvements can enhance problem-solving efficiency.

### Task 1: Basic Backtracking
- Solves Sudoku using brute-force recursion
- Serves as a baseline method

### Task 2: Smart Backtracking
- Uses MRV (Minimum Remaining Values)
- Uses LCV (Least Constraining Value)
- Solves puzzles faster than naive backtracking

### Task 3: Performance Analysis
- Compares solving time and steps for easy, medium, and hard Sudoku boards

## Project Structure
- `task1/` → Basic backtracking
- `task2/` → Smart backtracking with heuristics
- `task3/` → Step counter and timed comparisons
- `docs/` → Analysis PDF
